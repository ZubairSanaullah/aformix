## 📞 Support & Maintenance

### If You Need To...

**Change the email address:**
```ts
// src/constants/termsData.ts
contactInfo: {
  email: "newemail@aformix.com"  // Update here
}
```

**Add a new section:**
```ts
// Add to sections array in termsData.ts
{
  id: "new-section",
  title: "New Section Title",
  icon: "IconName",
  content: "Your content here..."
}
```

**Change colors:**
```tsx
// In TermsOfService.tsx, replace:
// from-blue-500 → from-red-500
// to-purple-500 → to-orange-500
// bg-blue-500 → bg-red-500
```

**Update "Last Updated" date:**
```ts
// src/constants/termsData.ts
lastUpdated: "January 2025"  // Update here
```
